# Secure Script

Cipher a script to share it

```
'aSecretKey' SECUREKEY
<S
  [
    'TOKEN'
    '~metric.*'
    { }
    NOW
    -1
  ] FETCH
S>
```

And to execute it
```
"secureScriptString" EVALSECURE
```
